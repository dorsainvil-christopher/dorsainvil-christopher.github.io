PROJET GamePastry

GamePastry est une application de jeu permettent de gagner des pâtisseries. l'application contient 2 pages: Une pour jouer, une pour affiché les résultats

RÈGLES DU JEU

Les utilisateurs doivent lancer 5 dés, s'ils obtiennent une combinaison gagnante, ils remportent des pâtisseries en fonction de la combinaison. L'utilisateur pourra rejouer autant de fois qu'il le souhaite. Voici les combinaisons gagnantes :

  • Un Yams : Gagne 4 pâtisseries
  • Un Carré : Gagne 3 pâtisseries
  • La grande suite : Gagne 1 pâtisserie

BASE DE DONNÉES

Nous avons créer une base de données à l'aide de PHPMyAdmin en établissant des liaisons entre elles avec la commande Symfony make:entity, puis en choisissant les relations

SYMFONY

Nous avons mis en place le framework Symfony qui nous sert a accéler la création et la maintenance des applications afin d'accéder aux informations souhaitées

  • Entity : Permet de manipuler les tables de la base de données
  • Controller : Sert à transmettre à la vue (template) toutes les données de la base de données provenant des repositories
  • Repository : Permet de récupérer des informations de la base de données
  • Service : Permet de mettre en place le fonctionnement du jeu

Pour créer ces informations, nous avons donc utilisé plusieurs commandes sur le terminal

Création d'Entités

bin/console make:entity
Commande pour créer des Entités. Dans notre cas, nous avons créé des objets Pastry et User

Création de Controller

bin/console make:controller
Commande pour créer des Contrôleurs, ce qui nous a permis de réaliser les messages des résultats obtenus

Création de Repositories

bin/console make:repository
Commande pour créer des Repositories, pour obtenir les informations de l'utilisateur

Création de Services

Ce fichier, qui n'a pas été créé par le terminal, détermine les actions attendues pour effectuer le jeu
