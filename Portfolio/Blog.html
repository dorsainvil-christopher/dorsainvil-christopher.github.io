PROJET GamePastry

GamePastry est une application de jeu permettent de gagner des pâtisseries. l'application contient 2 pages: Une pour jouer, une pour affiché les résultats

RÈGLE DE JEU

Les utilisateurs doivent lancée 5 dés, si ils obtiennent une combinaison gagnante, ils remporte des pâtisseries en fonction de la combinaison. L'utilisateur pourras rejouer autant de fois que l'on souhaite. Voici les combinaisons gagnantes

  • Un Yams : Gagne 4 pâtisseries
  • Un Carré : Gagne 3 pâtisseries
  • La grande suite : Gagne 1 pâtisserie

SYMFONY

Nous avons mis en place le framwork Symfony afin d'accéder au informations souhaité

  • Entity : Permet de manipuler des tables de la base de données
  • Controller : Sert à passer à la vue (template) toute les données de la base de donnés issue des repository
  • Repository : Permet de récuperer des infos de la base de données
  • Service : Permet de mettre en place le fonctionnement du jeu

Pour crée ces informations nous avons donc utilisé plusieur commande sur le terminal

Création Entity

bin/console make:entity
Commande pour créer des Entity, dans notre cas nous avons créer Pastry et User

Création Controller

bin/console make:controller
Commande pour créer des Controller,qui nous à permis de réaliser les messages des résultat obtenue

Création Repository

bin/console make:repository
Commande pour créer des Repository, pour avoir les informations de l'utilisateur

Création Service

Ce fichier qui n'a pas été crée par le terminal determine les actions attendu pour effectuer le jeu

BASE DE DONNÉES

Nous avons créer une Base de donnée à l'aide de PHPmyAdmin en faisant des liaison entre elles avec la commande symfony make:entity puis en choisissant les relations
